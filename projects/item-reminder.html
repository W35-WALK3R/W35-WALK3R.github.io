Item Reminder Tray

Overview: Arduino-powered tray using ultrasonic sensors and OLEDs to detect missing essentials.

My Role

Coded sensor logic, designed wiring diagrams, and created the OLED display layout.

Technical Details

  • Microcontroller: Arduino Nano
  • Sensors: HC-SR04 ultrasonic
  • Displays: 0.96″ OLED (128×64)
  • Features: Door sensor trigger + buzzer alert
